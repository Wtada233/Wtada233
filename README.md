### 我目前最有实力的项目：LankeOS

我正深潜于 **Linux From Scratch (LFS)** 的深渊。目标是将混乱的类似于Boost这种根本不遵守规范的包叠加在一起再加上我用AI写的包管理器改造成高性能、极简主义的极客乌托邦。

* **极速启动**: 从 UEFI 到图形界面 (Sway) 仅需 **3.128s**。（实体机测试，配置见下）
* **纯净血统**: 工具链 (**GCC 15.2 / Clang 21.1 / Rust 1.93**) 100% 源码构建。
* **完整环境**: Sway 窗口管理器，原生 Wayland，中文输入完美打通。
* **极致压缩**: 3.3GB 生产力环境压缩至 **644MB** ISO，静默内存占用仅 **~180MB**。
